gitfpush
========

Add and commit all the edited file to git with the commit message as the
variadic arguments without quotes and forcefully push the update to
remote

--------------

::

   File Name  : gitfpush.ps1
   Author     : Adewale Azeez - azeezadewale98@gmail.com
   Date       : Mar-01-2020

--------------

A short compact command to quickly add all edited files to a commit with
the commit messages and forcefully push the changes to the currently
active branch.

If your push is rejected you should merge or do what is right. Force
push to wreck havoc.

Syntax
------

.. code:: powershell

   gitfpush [[-args] <String[]>] [<CommonParameters>]

----------

.. code:: powershell

   Required?                    false

System.String[]

Outputs
-------

some text and analysis I don’t need

EXAMPLE 1
---------

::

   gitfpush from here is the commit message

your changes will be commit with the message “from here is the commit
message”. The command is equivalent to git add .; git commit -m “from
here is the commit message”; git push origin HEAD

EXAMPLE 2
---------

::

   gitfpush

This will commit your changes without any messages. git add .; git
commit -m ""; git push origin HEAD
