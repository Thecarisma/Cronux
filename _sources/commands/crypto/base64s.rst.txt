base64s
=======

Convert string to base64

--------------

::

   Version    : 1.0
   File Name  : base64.ps1
   Author     : Adewale Azeez - azeezadewale98@gmail.com
   Date       : 2020

--------------

Convert a string to it base64 value, write the base64 result into a
file, copy it to clipboard or generates a valid JSON Object.

Syntax
------

.. code:: powershell

   base64s [-InputString] <String> [-ToJson] 
   [-AndCopyToClipboard] [-WriteToFile] [<CommonParameters>]

Outputs
-------

Depends on switches

EXAMPLE 1
---------

::

   base64 "Normal String here" -WriteToFile

The command encodes the string in base64 and writes it to file
‘str.base64’ in working directory where this script was executed

EXAMPLE 2
---------

::

   base64s "Normal String here" -AndCopyToClipboard

The command encodes the string in base64 and copies the base64 value to
clipboard

EXAMPLE 3
---------

::

   base64 "Normal String here" -ToJson

The command encodes the string in base64 and generates a valid JSON
object of the base64 value
