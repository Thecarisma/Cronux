gitclone
========

A shorthand script to clone a repository master or specified branch

--------------

::

   File Name  : gitclone.ps1
   Author     : Adewale Azeez - azeezadewale98@gmail.com
   Date       : Mar-01-2020

--------------

A short compact command to quickly clone a repository. To clone a
specific branch add the branch name as the second parameter. The master
branch is clone by default if the branch name is not specified.

Syntax
------

.. code:: powershell

   gitclone [-RepoUrl] <String> [[-Branch] <String>] 
   [<CommonParameters>]  

Parameters
----------

.. code:: powershell

   Required?                    true

Outputs
-------

EXAMPLE 1
---------

::

   gitclone https://github.com/Thecarisma/Cronux.git

clone the Cronux repository The command is equivalent to git clone
https://github.com/Thecarisma/Cronux.git

EXAMPLE 2
---------

::

   gitclone https://github.com/Thecarisma/Cronux.git gh-pages

clone the Cronux repository branch ‘gh-pages’ The command is equivalent
to git clone -b gh-pages https://github.com/Thecarisma/Cronux.git
